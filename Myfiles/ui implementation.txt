VulnScope full-stack startup

From the vulnscope project directory, run:

docker compose up --build

Services:

- Frontend: http://localhost:5173
- Backend API: http://localhost:8000
- API documentation: http://localhost:8000/docs
- Authorized local test target: http://localhost:8081

Add targets through the VulnScope Targets page. Confirm authorization before scanning.

When VulnScope runs through Docker, a target entered as localhost or 127.0.0.1 is resolved through the host bridge automatically. All frontend screens read their data from the backend API.
